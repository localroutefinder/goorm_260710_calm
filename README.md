# Mycalm

잠들기 전 5분, 오늘 하루를 조용히 닫아요.

Mycalm은 잠이 부족한 한국 청소년이 자기 전 스마트폰을 내려놓고 수면 준비 루틴을 완료하도록 돕는 모바일 우선 웹앱입니다.

> **Mycalm은 의료 진단이나 치료를 제공하지 않습니다.**  
> 잠들기 전 긴장을 낮추고 수면 준비 루틴을 돕는 웹앱입니다.

## 로컬 실행

1. 이 저장소를 클론하거나 파일을 다운로드합니다.
2. `index.html`을 브라우저에서 엽니다. (더블클릭 또는 Live Server 사용)
3. 별도 설치나 빌드 과정이 필요 없습니다.

## 주요 기능

- 오늘 밤 방해 요인 선택 (생각, 시험, 폰, 긴장, 불면)
- 상태별 5분 수면 루틴 (호흡 → 이완 → 생각 정리)
- 단계별 타이머 및 진행률 표시
- 완료 기록 저장 (localStorage)
- 기록 조회 및 초기화
- 보호자/교사용 안내 화면

## 파일 구조

```
index.html    # 앱 진입점 (7개 화면 SPA)
style.css     # 스타일
app.js        # 로직 및 루틴 데이터
assets/       # 아이콘·이미지 (확장용)
```

## GitHub Pages 배포

1. GitHub에 저장소를 생성합니다.
2. 프로젝트 파일을 `main` 브랜치에 push합니다.
3. 저장소 **Settings → Pages**로 이동합니다.
4. **Source**: Deploy from a branch
5. **Branch**: `main`, **Folder**: `/ (root)`
6. Save 후 몇 분 뒤 `https://[사용자명].github.io/[저장소명]/` 에서 접속합니다.

`.nojekyll` 파일이 포함되어 있어 Jekyll 처리 없이 정적 파일이 그대로 배포됩니다.

## 개인정보 안내

### 수집하지 않는 정보

- 이름, 전화번호, 이메일
- 위치정보, 학교명
- 친구관계 내용, 건강 진단 정보

### 저장하는 정보 (브라우저 localStorage)

| 키 | 내용 |
|---|---|
| `mycalm_records` | 루틴 완료 기록 |
| `mycalm_last_state` | 마지막 선택 상태 |
| `mycalm_events` | 앱 사용 이벤트 로그 |

모든 데이터는 사용자의 브라우저에만 저장되며, 서버로 전송되지 않습니다.

## 기술 스택

- localStorage
- 프레임워크 없음, 빌드 도구 없음
