# Mycalm

잠들기 전 5분, 오늘 하루를 조용히 닫아요.

Mycalm은 잠이 부족한 한국 청소년이 자기 전 스마트폰을 내려놓고 수면 준비 루틴을 완료하도록 돕는 모바일 우선 웹앱입니다.

**라이브 데모:** https://localroutefinder.github.io/goorm_260710_calm/

> **Mycalm은 의료 진단이나 치료를 제공하지 않습니다.**  
> 잠들기 전 긴장을 낮추고 수면 준비 루틴을 돕는 웹앱입니다.

---

## 현재까지 구현 내용

### 1차 MVP (Vanilla HTML/CSS/JS SPA)

- 7개 화면 단일 페이지 앱: 홈 → 상태 선택 → 루틴 추천 → 루틴 진행 → 완료 → 기록 → 보호자 안내
- 5가지 상태별 5분 루틴 (3단계: 60초 + 120초 + 120초)
- 단계별 타이머, 진행률 바, localStorage 기록 저장
- 이벤트 로그 (`mycalm_events`) 및 기록 초기화

### 밤 퀘스트형 UX

- 미션 선택 → 3스테이지 퀘스트 맵 → 스테이지별 진행
- 타이머 종료 시 **보상 받기** → 별·XP 팝업 → 다음 미션
- 누적 성장: 레벨, XP, 별 (`mycalm_progress`)
- 퀘스트 클리어 보너스 (+20 XP, +2 별)

| 보상 | XP | 별 |
|---|---|---|
| 스테이지 1개 클리어 | +10 | +1 |
| 퀘스트 전체 클리어 보너스 | +20 | +2 |

### 시간대별 배경 + 반응형 레이아웃

- 6장 일러스트 배경이 **실제 시각**에 따라 자동 선택
- 60초마다 시각 재확인, 시간대 변경 시 크로스페이드 전환
- 전체 화면 배경 + 중앙 UI 컬럼 (모바일·태블릿·safe-area 대응)

## 주요 기능

- 밤 퀘스트형 UX — 미션 선택 → 3스테이지 클리어
- 단계 완료마다 별·XP 보상, 퀘스트 클리어 시 보너스
- 레벨·별 누적 성장 (localStorage)
- 상태별 5분 수면 루틴 (호흡 → 이완 → 생각 정리)
- 단계별 타이머 및 진행률 표시
- 퀘스트 기록 조회 및 초기화
- 보호자/교사용 안내 화면
- 시간대별 배경 일러스트 (실제 시각 기준, 60초마다 자동 갱신)

## 배경 이미지 (시간대별)

`assets/images/`에 6장의 일러스트가 있으며, **실제 시각**에 따라 자동으로 선택됩니다.

| 시간대 | 파일 |

| 05:00–08:59 | `bg-morning.png` |
| 09:00–12:59 | `bg-day.png` |
| 13:00–16:59 | `bg-afternoon.png` |
| 17:00–19:59 | `bg-sunset.png` |
| 20:00–22:59 | `bg-dusk.png` |
| 23:00–04:59 | `bg-night.png` |

## 로컬 실행

1. 저장소를 클론하거나 파일을 다운로드합니다.
2. `index.html`을 브라우저에서 엽니다. (더블클릭 또는 Live Server)
3. 별도 설치나 빌드 과정이 필요 없습니다.

## 파일 구조

```
goorm_260710_calm/
├── index.html          # 앱 진입점 (7개 화면 SPA)
├── style.css           # 스타일 (반응형 + 배경 레이어)
├── app.js              # 루틴·퀘스트·배경 로직
├── .nojekyll           # GitHub Pages 정적 배포
├── README.md
├── assets/
│   └── images/         # 시간대별 배경 6장
└── doc/
    └── MYCALM_TECH_SPEC.md
```

## GitHub Pages 배포

1. `main` 브랜치에 push합니다.
2. 저장소 **Settings → Pages** → Branch: `main`, Folder: `/ (root)`
3. https://localroutefinder.github.io/goorm_260710_calm/ 에서 접속합니다.

## 개인정보 안내

### 수집하지 않는 정보

| 키 | 내용 |
|---|---|
| `mycalm_records` | 루틴 완료 기록 |
| `mycalm_last_state` | 마지막 선택 상태 |
| `mycalm_events` | 앱 사용 이벤트 로그 |
| `mycalm_progress` | XP, 별, 레벨 진행도 |

모든 데이터는 사용자의 브라우저에만 저장되며, 서버로 전송되지 않습니다.
