# Mycalm 웹앱 기술명세서

## HTML / CSS / JavaScript 기반 수면·명상 MVP

---

## 1. 문서 목적

이 문서는 바이브코딩 입문자가 `HTML`, `CSS`, `JavaScript`만으로 **Mycalm 수면/명상 MVP 웹앱**을 만들기 위한 기술명세서이다.

Mycalm은 완성형 명상 플랫폼이 아니라, **잠이 부족한 한국 청소년이 자기 전 5분 동안 스마트폰을 내려놓고 수면 준비 루틴을 완료하는 가장 작은 웹앱**을 목표로 한다.

---

## 2. 서비스 개요

### 서비스명

**Mycalm**

### 서비스 형태

모바일 우선 웹앱

초기 버전은 앱스토어 출시 없이 브라우저에서 접속하는 방식으로 구현한다.

### 핵심 사용자

늘 잠이 부족한 한국 중·고등학생

### 핵심 문제

청소년 사용자는 잠들기 전 다음과 같은 이유로 수면 전환이 어렵다.

| 문제 | 설명 |
|---|---|
| 스마트폰 과사용 | 숏폼, 메신저, 게임 때문에 잠드는 시간이 늦어짐 |
| 학업 스트레스 | 시험, 수행평가, 성적 걱정 |
| 생각 과다 | 친구관계, 내일 일정, 후회, 불안 |
| 수면 루틴 부재 | 자기 전 일정한 마무리 행동이 없음 |

### 핵심 목적

사용자가 자기 전 **오늘 상태를 선택하고, 5분 수면 루틴을 실행한 뒤, 완료감을 느끼며 화면을 닫도록 돕는다.**

---

## 3. Mycalm 한 줄 정의

> Mycalm은 잠들기 전 5분 동안 청소년이 머릿속 소음을 줄이고 스마트폰을 내려놓게 돕는 수면 준비 웹앱이다.

---

## 4. MVP 핵심 흐름

```txt
1. 홈 화면 진입
2. 오늘 밤 방해 요인 선택
3. 추천 수면 루틴 확인
4. 5분 루틴 시작
5. 단계별 호흡·이완·생각 정리 진행
6. 완료 카드 확인
7. 완료 기록 저장
8. 기록 화면에서 사용 내역 확인
```

---

## 5. 1차 MVP 포함 기능

| 기능 | 설명 | 우선순위 |
|---|---|---|
| 홈 화면 | Mycalm 소개와 시작 버튼 | 필수 |
| 상태 선택 | 오늘 밤 잠을 방해하는 이유 선택 | 필수 |
| 루틴 추천 | 선택 상태에 맞는 5분 루틴 제공 | 필수 |
| 루틴 진행 화면 | 호흡, 몸 이완, 생각 정리 단계 진행 | 필수 |
| 타이머 | 각 단계별 남은 시간 표시 | 필수 |
| 완료 카드 | “오늘 잠 준비 완료” 메시지 표시 | 필수 |
| 기록 저장 | localStorage에 완료 기록 저장 | 필수 |
| 기록 보기 | 사용 횟수, 최근 완료일, 최근 루틴 표시 | 필수 |
| 기록 초기화 | 저장된 기록 삭제 | 권장 |
| 보호자/교사용 안내 | 앱 목적과 개인정보 최소 수집 설명 | 권장 |

## 6. 1차 MVP 제외 기능

| 푸시 알림 | 웹 입문 단계에서는 후순위 |

## 9. 화면 구성

### 9-1. 홈 화면

#### 목적

사용자가 앱의 목적을 이해하고 바로 루틴을 시작하게 한다.

#### 표시 요소

| 요소 | 내용 |
|---|---|
| 앱 이름 | Mycalm |
| 한 줄 설명 | 잠들기 전 5분, 오늘 하루를 조용히 닫아요. |
| 시작 버튼 | 오늘 밤 루틴 시작하기 |
| 기록 버튼 | 내 수면 준비 기록 보기 |
| 안내 링크 | 보호자/교사용 안내 |

#### 예시 문구

```txt
오늘도 잠들기 어려웠다면,
5분만 같이 정리해볼게요.
```

#### 주요 동작

| 사용자 행동 | 동작 |
|---|---|
| 시작 버튼 클릭 | 상태 선택 화면으로 이동 |
| 기록 버튼 클릭 | 기록 화면 표시 |
| 안내 링크 클릭 | 보호자/교사용 안내 화면 표시 |

---

### 9-2. 상태 선택 화면

#### 목적

사용자가 지금 잠을 방해하는 이유를 선택하게 한다.

#### 선택지

| 상태 ID | 버튼 문구 | 추천 루틴 |
|---|---|---|
| `too_many_thoughts` | 생각이 많아요 | 머리 끄기 루틴 |
| `exam_stress` | 시험이 걱정돼요 | 시험 전날 내려놓기 루틴 |
| `phone_overuse` | 폰을 못 내려놔요 | 화면 닫기 루틴 |
| `body_tension` | 몸이 긴장돼요 | 몸 힘 빼기 루틴 |
| `just_awake` | 그냥 잠이 안 와요 | 조용히 잠드는 루틴 |

#### 예시 문구

```txt
오늘 밤, 뭐가 제일 방해돼?
```

#### 주요 동작

| 사용자 행동 | 동작 |
|---|---|
| 상태 버튼 클릭 | 선택값 저장 |
| 상태 선택 완료 | 루틴 추천 화면으로 이동 |

---

### 9-3. 루틴 추천 화면

#### 목적

선택한 상태에 맞는 5분 루틴을 보여준다.

#### 표시 요소

| 요소 | 설명 |
|---|---|
| 추천 루틴 제목 | 예: 머리 끄기 5분 루틴 |
| 루틴 설명 | 짧은 안내 문장 |
| 총 소요 시간 | 5분 |
| 단계 요약 | 호흡 1분, 몸 이완 2분, 생각 정리 2분 |
| 시작 버튼 | 5분 루틴 시작 |
| 뒤로가기 버튼 | 상태 다시 선택 |

#### 예시 문구

```txt
오늘은 생각을 없애려 하지 말고,
잠시 옆에 내려놓는 연습만 해볼게요.
```

---

### 9-4. 루틴 진행 화면

#### 목적

사용자가 5분 루틴을 단계별로 진행하게 한다.

#### 기본 단계

| 단계 | 시간 | 내용 |
|---|---:|---|
| 1단계 | 60초 | 천천히 호흡하기 |
| 2단계 | 120초 | 어깨, 턱, 손 힘 빼기 |
| 3단계 | 120초 | 내일 걱정은 내일로 넘기기 |

#### 표시 요소

| 요소 | 설명 |
|---|---|
| 현재 단계명 | 예: 1단계. 숨 고르기 |
| 안내 문구 | 짧고 안정적인 문장 |
| 남은 시간 | 초 단위 카운트다운 |
| 진행 바 | 전체 루틴 진행률 |
| 다음 단계 버튼 | 타이머 완료 시 활성화 |
| 그만하기 버튼 | 홈으로 이동 |

#### 주요 동작

| 조건 | 동작 |
|---|---|
| 단계 시작 | 타이머 시작 |
| 시간이 끝남 | 다음 단계 버튼 활성화 |
| 다음 단계 클릭 | 다음 단계 표시 |
| 마지막 단계 완료 | 완료 화면으로 이동 |
| 그만하기 클릭 | 홈 화면으로 이동 |

---

### 9-5. 완료 카드 화면

#### 목적

사용자에게 작은 완료감을 준다.

#### 표시 요소

| 요소 | 내용 |
|---|---|
| 완료 메시지 | 오늘 잠 준비 완료 |
| 부드러운 문장 | 이제 화면은 닫아도 괜찮아. |
| 완료 횟수 | 지금까지 완료한 루틴 수 |
| 홈 버튼 | 처음으로 |
| 기록 보기 버튼 | 기록 화면으로 이동 |

#### 완료 시 저장 데이터 예시

```js
{
  id: 1720610100000,
  date: "2026-07-10",
  routineId: "too_many_thoughts",
  completed: true,
  completedAt: "2026-07-10T23:15:00"
}
```

---

### 9-6. 기록 화면

#### 목적

사용자가 자신의 루틴 사용 기록을 확인하게 한다.

#### 표시 요소

| 요소 | 설명 |
|---|---|
| 전체 완료 횟수 | 총 몇 번 완료했는지 |
| 최근 완료일 | 마지막 사용 날짜 |
| 최근 루틴 | 최근 선택한 루틴 |
| 기록 목록 | 최근 완료 기록 리스트 |
| 기록 초기화 버튼 | localStorage 삭제 |

#### 예시 문구

```txt
이번 주에는 3번,
잠들기 전 멈추는 연습을 했어요.
```

---

### 9-7. 보호자/교사용 안내 화면

#### 목적

미성년자 대상 앱이므로 보호자와 교사가 안심할 수 있도록 앱 목적을 설명한다.

#### 필수 문구

```txt
Mycalm은 청소년의 수면 준비 루틴을 돕기 위한 웹앱입니다.
의료 진단이나 치료를 제공하지 않습니다.

초기 버전에서는 이름, 전화번호, 위치정보를 수집하지 않습니다.
사용 기록은 사용자의 브라우저 안에만 저장됩니다.
```

#### 표시 요소

| 요소 | 설명 |
|---|---|
| 앱 목적 | 수면 전환 루틴 |
| 수집 정보 | localStorage 기록 |
| 수집하지 않는 정보 | 이름, 연락처, 위치 |
| 사용 대상 | 청소년과 보호자 |
| 홈 버튼 | 처음으로 이동 |

## 26. 최종 판단 기준

Mycalm 1차 MVP의 핵심 질문은 다음 하나이다.

> 청소년 사용자가 자기 전 5분 루틴을 실제로 시작하고 끝까지 완료하는가?
